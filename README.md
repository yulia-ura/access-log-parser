# Парсер серверных access-логов

Парсер серверных access-логов

Учебный проект на Java.

Реализованы:
- чтение логов из файла;
- обработка данных с использованием Stream API;
- расчёт статистики по запросам;
- поиск наиболее посещаемых страниц;
- анализ кодов ответа сервера.

Технологии:
- Java
- Stream API
- Maven
